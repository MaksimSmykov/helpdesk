# IT Navigator — инструкция для сотрудника
Статический сайт (HTML/CSS/JS без сборки). Все данные хранятся в `localStorage` браузера.
IT Navigator помогает быстро решить IT-вопрос: сначала предлагает проверенные шаги и статьи, а если проблема не решилась — помогает создать заявку для специалиста.
## Структура
## Как начать
- `index.html` — лендинг
- `pages/` — страницы приложения (диагностика, заявки, база знаний, услуги, миссии, профиль, IT-панель)
- `css/` — стили (`variables`, `base`, `components`, `animations`, `pages`)
- `js/` — логика (`data`, `navigation`, `tickets`, `profile`, `diagnostics`, `missions`, `app`)
- `assets/` — иконки и изображения
Откройте главную страницу IT Navigator. Вы попадёте в личный кабинет сотрудника, где видны:
## Как запустить локально
- ваши контактные данные;
- рабочее место;
- ноутбук и подключённые устройства;
- открытые обращения;
- цифровые навыки;
- быстрые действия для решения IT-вопросов.
Нужен любой статический веб-сервер (просто открыть `index.html` через `file://` может не работать из-за ограничений браузера на `fetch`/пути).
## Быстрые действия
### Вариант 1 — Node.js
На главной странице доступны основные кнопки:
```bash
npx serve .
```
- `Проверить проблему` — запустить короткую диагностику;
- `Оставить заявку` — сразу создать обращение в IT;
- `База знаний` — найти инструкцию по частой проблеме;
- `История обращений` — посмотреть прошлые и текущие заявки.
### Вариант 2 — Python
## Если возникла проблема
```bash
python -m http.server 8080
```
1. Нажмите `Нужна помощь` или `Проверить проблему`.
2. Выберите категорию: вход, интернет/VPN, устройство, программа, доступ, установка ПО или другая проблема.
3. Ответьте на уточняющие вопросы.
4. Выполните предложенные проверки.
5. Если проблема решена — нажмите `Проблема решена`.
6. Если не помогло — нажмите `Передать специалисту`.
После запуска откройте адрес, который покажет команда (обычно `http://localhost:3000` или `http://localhost:8080`).
После передачи специалисту заявка будет заполнена автоматически: категория, ответы, проверки, устройство и предполагаемая причина попадут в описание.
## Как создать заявку вручную
Используйте кнопку `Оставить заявку`, если вы уже понимаете, что нужна помощь IT.
В заявке укажите:
- тему;
- категорию;
- устройство;
- описание проблемы;
- дополнительный комментарий;
- файл или скриншот, если он поможет специалисту.
После отправки заявка появится в истории обращений.
## База знаний
В разделе `База знаний` можно искать инструкции по ключевым словам: VPN, пароль, Outlook, Teams, браузер, доступы и другим темам.
У каждой статьи есть оценка:
- `Полезно` — если инструкция помогла;
- `Не помогло — создать заявку` — если нужно подключить специалиста.
## Навыки
Раздел `Навыки` показывает цифровую самостоятельность сотрудника.
Примеры навыков:
- безопасный вход с двухфакторной авторизацией;
- удалённый доступ через VPN;
- актуальный браузер;
- корпоративная почта.
Нажмите на навык, чтобы открыть боковую панель. В ней отображаются:
- описание навыка;
- выбранный вариант настройки;
- прогресс;
- список шагов;
- ссылки на инструкции в базе знаний;
- кнопка создания заявки, если шаг выполнить не получается.

- ФИО и отдел;

- email и телефон;
- рабочее место;

## Важно
IT Navigator не заменяет специалиста. Он помогает быстрее собрать данные и решить типовые вопросы самостоятельно. Если предложенные шаги не помогли, заявка будет передана IT-отделу с уже заполненным контекстом.
